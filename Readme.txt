Intrucciones para ejecutar la aplicación:
 1. Crear ambiente virtual con el comando: python3 -m venv venv 
 2. Inicializar el ambiente virtual (Linux-macos): . venv/bin/activate  
 2. Inicializar el ambiente virtual (Windows): .\venv\Scripts\activate 
 3. Instalar dependencias: pip install -r requirements.txt
 4. Correr el programa: streamlit run index.py
 5. Para abrir el programa, ir al siguiente link: http://localhost:8501
